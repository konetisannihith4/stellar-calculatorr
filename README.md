# Math Ink

A Vite + React app for the `konetisannihith4/math-ink` repository.

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/7712b631-6c5e-4ee8-8557-d5b03ce9f7ef

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
